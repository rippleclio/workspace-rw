# Wabifair Design System

> Generated by /design-consultation — 2026-04-11
> Status: APPROVED (all three design risks accepted)

## Product Context

Wabifair 是高端品牌电商平台，服务于对东亚审美有认同感的买家。平台连接独立创作者品牌与全球消费者，覆盖中国、泰国、新加坡、日本、韩国市场。

技术栈：React 18 + TypeScript + Vite + TailwindCSS + CSS Custom Properties。

**核心设计命题：** 东亚美学是根，不是附件。

---

## Aesthetic Direction — 精炼侘び（Refined Wabi）

品牌美学关键词：侘寂（wabi-sabi）、减法、留白、自然缺陷之美、时间感。

竞品参考（调研于 2026-04-11）：
- **Loewe** — 最少 5 项导航，品牌名即 h1，极度留白
- **Bottega Veneta** — 全出血视频 hero，视觉上的沉静自信
- **Dover Street Market** — 文化机构叙事，不做促销语言

**EUREKA 洞察（来自竞品调研）：** 所有西方奢品都用西方视觉语言。Wabifair 的差异化是把东亚美学作为根底，而不是主题覆盖层。侘寂哲学从首屏开始，不是从"中文语言包"开始。

### 设计原则

1. **减法优先** — 删除所有非必要元素，直到删无可删
2. **不对称平衡** — 避免居中堆砌，偏移构图更有张力
3. **材质感** — 文字排版本身是质感，不依赖装饰图形
4. **沉默的奢华** — 不喊"高端"，通过留白和字重传递
5. **东亚时间感** — 汉字与拉丁字母共存，且汉字优先

### Spinner 出现延迟规则

所有 loading spinner 必须在 200ms 后才出现，防止快速响应场景的闪烁：

```typescript
// useDelayedLoading.ts
export function useDelayedLoading(isLoading: boolean, delay = 200) {
  const [showSpinner, setShowSpinner] = useState(false);
  useEffect(() => {
    if (!isLoading) { setShowSpinner(false); return; }
    const t = setTimeout(() => setShowSpinner(true), delay);
    return () => clearTimeout(t);
  }, [isLoading, delay]);
  return showSpinner;
}
```

## Dark Mode

`data-mode="dark"` 属性控制，JS 写入 `document.documentElement`。底色用 `secondary-900` 暖棕 #1A1714（不用冷灰），保持侘寂温度感。

### Token Override

```css
[data-theme="apple"][data-mode="dark"] {
  /* 页面底色：暖棕 secondary-900，不用冷灰 Apple Dark BG */
  /* body { background-color: rgb(var(--color-page-bg)) } */
  --color-page-bg:          26  23  20;   /* #1A1714 — 暖棕，保持侘寂质感 */
  --color-bg:               28  28  30;   /* #1C1C1E — Apple Dark BG，面板内部 */
  --color-surface:          44  44  46;   /* #2C2C2E — Apple Elevated，卡片底 */
  --color-text:            245 245 247;   /* #F5F5F7 — Primary Text */
  --color-text-secondary:  134 134 139;   /* #86868B — Secondary（与浅色一致）*/
  --color-text-muted:       99  99 102;   /* #636366 — Muted */
  --color-border:           56  56  58;   /* #383838 — Dark Border */
  --color-border-hover:     72  72  74;   /* #48484A — Dark Hover Border */
  /* accent: 196 162 78 / #C4A24E — 暗色底对比度约 7:1，完全满足 WCAG AA */
```

### JS 实现（`src/utils/theme.ts`）

```typescript
type ColorMode = 'light' | 'dark' | 'system';

export function setColorMode(mode: ColorMode) {
  const root = document.documentElement;
  if (mode === 'system') {
    root.removeAttribute('data-mode');
  } else {
    root.setAttribute('data-mode', mode);
  }
  localStorage.setItem('wabifair-color-mode', mode);
}

export function initColorMode() {
  const saved = localStorage.getItem('wabifair-color-mode') as ColorMode | null;
  if (saved === 'dark' || saved === 'light') {
    setColorMode(saved);
  }
  // system: let prefers-color-scheme handle it via CSS
}
```

### 字重视觉补偿

韩文字符视觉重量比同等 Latin 字重偏重约 1 级：

| 场景 | Latin 字重 | 对应韩文字重 |
|------|-----------|-------------|
| 正文 body | 400 | 400（Noto KR 的 400 已较轻）|
| UI labels | 500 | 400 |
| Heading h3 | 600 | 500 |
| Heading h1/h2 | 700 | 600 |

不要用 300（too thin，低分辨率屏幕可读性差）。不要用 700（韩文 700 视觉过重，破坏侘寂减法原则）。

### Logo / Brand Name 处理

Logo 用 Noto Serif SC（zh-CN 场景）在 ko-KR locale 下无汉字，品牌名 "Wabifair" 是英文。建议：
- ko-KR 时 Logo 字体改为 Plus Jakarta Sans 600（拉丁显示清晰）
- 不强制使用 Noto Serif SC（ko-KR 衬线字体与 Noto Serif SC 无关联）

实现：`[lang="ko"] .brand-logo { font-family: var(--font-display); }`
